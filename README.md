# News Management System
A desktop application built with Python and Tkinter for managing news articles and user information with a MySQL database backend.

 ## Features
 ### User Management
* Add, update, and delete users
* View all registered users in a scrollable table
* View all news articles posted by a specific user
* Clean and intuitive user interface

### News Management
* Create news articles with titles and body content
* Update existing news articles
* Delete news articles
* View news articles with author information
* Full news preview with detailed content display
* Automatic timestamp tracking for each article

### Database Features
* Automatic database and table creation
* Sample data generation for testing
* Foreign key relationships between users and news
* Cascade delete functionality

## Technologies Used
* Python - Core programming language
* Tkinter - GUI framework
* MySQL Connector - Database connectivity
* MySQL - Database management system
